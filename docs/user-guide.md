## Getting Started

### Downloading the App

**For iPhone (iOS)**:
1. Open the **App Store**
2. Search for "Jomla Grocery Store"
3. Tap **Get** to download
4. Open the app

**For Android**:
1. Open **Google Play Store**
2. Search for "Jomla Grocery Store"
3. Tap **Install**
4. Open the app

### First Launch

When you first open the app:
1. You'll see the welcome screen
2. Tap **"Get Started"** to create an account
3. Or tap **"Sign In"** if you already have an account

## Creating Your Account

### Registration

1. **Enter Your Information**:
   - First Name
   - Last Name
   - Email Address
   - Phone Number (with country code)
   - Password (at least 8 characters)

2. **Verify Your Phone Number**:
   - You'll receive a 6-digit code via SMS
   - Enter the code in the app
   - Tap **"Verify"**

3. **Start Shopping**!
   - Your account is now active
   - You can browse and order

**Tips**:
- Use a valid email - you'll receive order confirmations
- Use a phone number you have access to
- Choose a strong password

### Verification Code Issues

**Didn't Receive Code?**:
- Wait up to 30 seconds
- Check your phone has signal
- Tap **"Resend Code"** (max 3 times per hour)

**Code Expired?**:
- Codes are valid for 30 minutes
- Request a new code

## Shopping Cart

### Adding Items to Cart

**From Offers**:
1. Tap an offer
2. Select quantity using **+** and **-** buttons
3. Tap **"Add to Cart"**
4. See confirmation message

**From Products**:
1. Tap a product
2. Select quantity
3. Tap **"Add to Cart"**

**Quantity Limits**:
- Some items have minimum quantities (e.g., min 2)
- Some items have maximum quantities (e.g., max 5)
- App will show limits when you select quantity

### Viewing Your Cart

1. Tap the **Cart** icon at the bottom
2. See all items you've added:
   - **Offers** section: Bundled offers
   - **Individual Products** section: Single items

**Cart Summary Shows**:
- Subtotal
- Total Savings (from offers)
- Estimated Delivery Fee (if applicable)
- **Grand Total**

### Editing Cart

**Change Quantity**:
- Tap **+** or **-** next to any item
- Updates automatically

**Remove Items**:
- Swipe left on any item
- Tap **"Remove"**
- Or tap the trash icon

**Empty Cart**:
- All items remain until you remove them
- Cart saves even if you log out

## Placing Orders

### Checkout Process

1. **Review Your Cart**:
   - Make sure all items are correct
   - Check quantities
   - Note the total price

2. **Tap "Checkout"**

3. **Choose Fulfillment Method**:

   **Option 1: Delivery**
   - Tap **"Delivery"**
   - Enter delivery address:
     - Street address
     - City
     - Postal code
     - Delivery notes (optional)
   - See delivery fee added to total

   **Option 2: Pickup**
   - Tap **"Pickup"**
   - Select pickup time
   - See pickup location
   - No delivery fee

4. **Review Order Summary**:
   - All items listed
   - Offer products shown in detail
   - Delivery/pickup information
   - Final total

5. **Place Order**:
   - Tap **"Place Order"**
   - Order is submitted
   - You'll see confirmation screen

6. **Order Confirmation**:
   - Order number (e.g., ORD-20251107-0001)
   - Expected delivery/pickup time
   - Invoice download link
   - Track order button

**Payment**:
- Currently: Payment on delivery/pickup
- Future: Card payment in app

### Order Confirmation

After placing order, you'll receive:
- **In-App Notification**: Order confirmed
- **Email**: Order details and invoice
- **SMS**: Order status updates (optional)

## Order Tracking

### Viewing Your Orders

1. Tap **Profile** tab
2. Tap **"My Orders"**
3. See list of all your orders:
   - Most recent first
   - Order number
   - Date
   - Total amount
   - Current status

### Order Status

Your order progresses through these stages:

**1. Pending**:
- Order received by store
- Awaiting confirmation

**2. Confirmed**:
- Store confirmed your order
- Being prepared

**3. Preparing**:
- Items being gathered and packed

**4. Out for Delivery** (Delivery orders):
- Order is on the way to you
- Track estimated time

**5. Ready for Pickup** (Pickup orders):
- Order ready at store
- You can pick it up

**6. Completed**:
- Order delivered or picked up
- Thank you!

### Order Details

- Complete item list

## Troubleshooting

### Common Issues

#### Can't Log In

**Problem**: "Invalid email or password"
**Solution**:
- Check email is typed correctly
- Check Caps Lock is off
- Try **"Forgot Password"**
- Contact support if issue persists

#### Didn't Receive Verification Code

**Solutions**:
- Wait up to 30 seconds
- Check phone has cell signal
- Tap **"Resend Code"**
- Make sure phone number is correct with country code

#### App is Slow

**Solutions**:
- Close and restart the app
- Check internet connection
- Clear app cache (in phone settings)
- Update to latest app version

#### Offer Not Showing Correctly

**Possible Causes**:
- Offer expired
- Offer was modified by store
- Try refreshing: Pull down on screen

**Solution**:
- Remove from cart and re-add
- Contact support if problem continues

#### Can't Add Item to Cart

**Check**:
- Item is in stock
- You haven't exceeded max quantity
- You've met minimum quantity
- Item is active

#### Order Not Updating

**Try**:
- Pull down to refresh order status
- Check internet connection
- Wait a few minutes and check again
- Contact store if urgent

#### Payment Issues (When Implemented)

**For Card Declined**:
- Check card details are correct
- Ensure sufficient funds
- Contact your bank
- Try another card

## FAQs

**Q: Is there a minimum order amount?**
A: No, but delivery fee applies to all delivery orders.

**Q: Can I modify an order after placing it?**
A: Contact support immediately. Changes may be possible if order hasn't been prepared.

**Q: How do I cancel an order?**
A: Contact support as soon as possible. Cancellation depends on order status.

**Q: Are prices the same as in-store?**
A: Yes, plus delivery fee for delivery orders.

**Q: What if an item is out of stock?**
A: We'll contact you with substitution options or refund that item.

**Q: Can I schedule orders in advance?**
A: Yes, select future pickup time or delivery window.

**Q: What payment methods are accepted?**
A: Currently cash on delivery/pickup. Card payment coming soon.

**Q: Is my information secure?**
A: Yes, we use industry-standard encryption and security practices.
